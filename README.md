### 1.简介
币付钱包APP是一款移动端数字资产钱包，专注于多链多币种管理，为用户提供安全、简单、易用的数字资产管理服务。本项目设计模式采用MVC，第三方库使用cocopods管理，开发者下载之后，pod install即可运行.

### 2.功能模块
#### 2.1 钱包管理
* 创建钱包
* 导入钱包（可使用助记词导入或者私钥导入）
* 切换钱包
* 钱包详情
#### 2.2 币种说明
* 添加资产（添加资产后首页会显示出添加的币种）
* 余额，交易记录展示
* 交易记录详情展示
* 转账
* 收款
#### 2.3 兑币
* 兑换币种（通过changelly平台兑换）

### 3.币种说明
币付钱包暂时支持币种：BTC、BCH、ETH、LTC、USDT、XNE
